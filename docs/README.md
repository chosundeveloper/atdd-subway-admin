## 요구사항

### 이름

- [x] 이름 생성
    - [x] null 이거나 empty일 수 없다.

### 구간

#### 구간

- [x] 구간 생성
    - [x] 라인을 갖는다.
    - [x] 상행역을 갖는다.
    - [x] 하행역을 갖는다.
    - [x] 거리를 갖는다.
        - [x] 거리는 0보다 커야한다.

#### 구간들

- [x] 구간을 추가한다.
- [x] 역 목록을 조회한다.

### 노선
-
- [x] 색상 생성
    - [x] 색상은 null 이거나 empty일 수 없다.
- [x] 지하철 노선 생성
    - [x] 이름을 갖는다.
    - [x] 색상을 갖는다.
    - [x] 노선 생성시 역이 없을 경우 NOT_FOUND 응답 코드를 반환한다.
- [x] 노선 수정
    - [x] 이름을 수정한다.
    - [x] 색상을 수정한다.
- [x] 지하철 노선 조회
- [x] 지하철 노선 삭제
- [x] 지하철 노선 목록 조회
